# TCP/IP_IOT

# 얼굴 표정 인식

라즈베리파이를 활용해 TCP/IP 통신으로 서버와 클라이언트를 구축하는 역할과 전체적인 센서 제어를 담당하였습니다. 불꽃 감지 센서, 온습도 센서의 데이터에 따라 서보모터, LED, 버저가 동작합니다.

수행기간: 2023.11.13~2024.11.17
